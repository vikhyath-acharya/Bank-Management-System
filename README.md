🏦 Bank Management System

A console-based Bank Management System developed in C that demonstrates file handling, structures, and modular programming. The application allows users to manage bank accounts efficiently by storing account information in a binary file.

📌 Features

- Create a new bank account
- Display all accounts
- Search an account by account number
- Deposit money
- Withdraw money
- Delete an account
- Persistent data storage using binary files

🛠️ Technologies Used

- C Programming
- Standard C Libraries
- File Handling (Binary Files)

📂 Project Structure

Bank-Management-System/
│── bank.c
│── bank.dat        (Generated automatically)
└── README.md

▶️ How to Run

1. Clone the repository:

git clone https://github.com/vikhyath-acharya/Bank-Management-System.git

2. Compile the program:

gcc bank.c -o bank

3. Run the executable:

./bank

For Windows:

bank.exe
